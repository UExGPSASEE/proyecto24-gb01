# Mediflí

## Despliegue

En el directorio raíz del proyecto, para crear las imágenes:  

```bash
docker compose build
```

Para desplegar los contenedores:  

```bash
docker compose up -d
```

Para parar los contenedores:

```bash
docker compose down
```
